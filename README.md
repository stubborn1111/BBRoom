## 项目结构图

>*文件为配置文件，尽量不要改动

```
|-- /.idea/           #*idea配置
|-- /log/              #日志文件
|-- /project_document/ #项目文档
|-- /src/
|  |--/test/          #存放java测试代码
|  |-- /java/         #存放java源码
|  |-- /resources/    #存放项目配置文件
|  |-- /webapp/       #前端源码及以下
|     |-- /WEB-INF/
|        |-- web.xml   #webapp配置文件
|-- /target/           #存放编译生成文件
|-- /BBRoom.sql/       #数据库文件 
|-- pom.xml            #maven配置信息
|-- README.md          #项目简介
```
